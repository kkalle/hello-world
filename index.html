Welcome to ShootingApp project!

ShootingApp is a going to be an application that lets you record and analyze your target shooting scores. The app is developed by hobbyists for their own and general use.

The app is going to have the following features:

  • Mobile app for recording scores and basic analysis
  • Web application for deeper analysis
  • A scoreboard for sharing records at different locations

We plan to first implement the mobile app for iOS/iPhone and later release version for Android and Windows Phone.
